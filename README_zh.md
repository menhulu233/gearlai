# 鉴玄

**Gearl** — 基于 Claude 的桌面 AI Agent 应用，由 Electron 驱动。

配置好 API Key，选择一个 Agent 角色，开始协作。鉴玄替你完成任务，而你始终掌控一切。

## 核心特性

- **Agent 系统** — 自定义角色提示词与技能配置
- **多 Agent 会话** — 每个 Agent 拥有独立的会话历史
- **技能与 MCP** — 通过 Skills 和 MCP 服务器扩展能力
- **记忆** — 跨会话持久化用户偏好与上下文
- **Artifacts 预览** — HTML、SVG、Mermaid、React 实时预览
- **跨平台** — macOS、Windows、Linux

## 快速开始

```bash
git clone https://github.com/menhulu233/gearl.git
cd gearl
npm install
npm run electron:dev
```

## 环境要求

- Node.js >= 24 < 25
