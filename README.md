# Gearl

**鉴玄** — A desktop AI Agent powered by Claude, built with Electron.

Configure your API key, pick an Agent persona, and collaborate. Gearl handles tasks — coding, research, document creation — while you stay in control.

## Features

- **Agent System** — Define personas with custom prompts and skill sets
- **Multi-Agent Sessions** — Each Agent has its own isolated conversation history
- **Skills & MCP** — Extend capabilities with Skills and Model Context Protocol servers
- **Memory** — Persistent user context across sessions
- **Artifacts** — Live preview for HTML, SVG, Mermaid, React
- **Cross-Platform** — macOS, Windows, Linux

## Quick Start

```bash
git clone https://github.com/menhulu233/gearl.git
cd gearl
npm install
npm run electron:dev
```

## Requirements

- Node.js >= 24 < 25
